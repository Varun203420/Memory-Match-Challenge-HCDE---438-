# Memory-Match-Challenge-HCDE---438-

Memory Match Challenge is a browser-based card matching game where players flip two cards at a time to find matching pairs.
The game includes a timer, move tracking, difficulty selection, and a results summary screen.
This project was developed as part of the HCDE 438 – Interactive Web Components final project.

The app is built using React + Vite and demonstrates component-based design, React hooks, and state management.

Features Implemented
Core Functionality:
1) Start screen (Home)

2) Game screen with:
   Timer using useEffect
   Grid of interactive card components
   Quit + Finish buttons
   Difficulty selection (Easy / Medium / Hard)

4) Results screen showing:
   Total time
   Moves (placeholder)
   Matched pairs (placeholder)

5) Component-based structure (Card, Home, Game, Results)

6) Working navigation between pages using React state

7) Basic styling
